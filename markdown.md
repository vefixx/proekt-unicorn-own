# Примечание
Markdown изначально задуман как надстройка над HTML - текст на Markdown преобразуется в HTML, но любой валидный HTML, вставленный прямо в документ, остаётся нетронутым. Это значит, что можно использовать практически любые теги HTML.

## Выделение текста
- *Курсив*:	`*текст*` или `_текст_`
- **Полужирный**:	`**текст**` или `__текст__`
- ***Полужирный курсив***:	`***текст***` или `___текст___`
- ~~Зачёркнутый~~:	`~~текст~~`	`~~текст~~`

- Пункт 1

- Пункт 3
```

3. Третий пункт

## Гиперссылки
```

```

---

Ссылка в виде сноски:
```

[метка]: https://example.com "опционально"
```

[метка]: https://example.com "опционально"

---

```

<email@example.com>
```
